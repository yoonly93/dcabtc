# BTC 목표 투자 계산기

비트코인 목표 투자 금액과 기간을 계산할 수 있는 간단한 웹 시뮬레이션입니다.  
빗썸의 BTC/원 시세를 기준으로 매월 상승률을 적용하여 목표 BTC까지 필요한 기간과 총 투자 금액을 계산합니다.

## 기능
- 목표 비트코인 개수 입력
- 월별 투자금액 입력 (+10만, +100만, +1000만 버튼 포함)
- 월별 예상 상승률 조정 (기본 5%)
- 필요한 투자 기간과 총 투자 금액 계산
- 월별 투자 내역 테이블 표시

## 사용 방법
1. `index.html` 파일을 브라우저에서 열거나, Vercel/GitHub Pages로 배포
2. 목표 BTC, 월별 투자금액, 상승률 입력
3. **확인하기** 버튼 클릭
4. 결과 및 월별 내역 테이블 확인

## 주의사항
- 이 계산은 BTC 가격이 매월 설정한 %만큼 꾸준히 상승한다고 가정합니다.
- 실제 결과를 보장하지 않습니다.
- 교육적 목적으로만 사용하세요.
